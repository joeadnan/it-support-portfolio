# IT Support Portfolio

Website portfolio pribadi untuk menampilkan kemampuan, project, studi kasus, SOP, checklist, dan dokumentasi teknis di bidang IT Support.

## Tentang Project

Project ini dibuat menggunakan React, TypeScript, Vite, dan Tailwind CSS. Website ini digunakan sebagai portfolio profesional untuk menampilkan kemampuan IT Support, mulai dari troubleshooting hardware, software, jaringan, deployment komputer, asset inventory, backup, CCTV, hingga dokumentasi teknis.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- HTML
- CSS

## Fitur Portfolio

- Hero section profesional
- Skill IT Support
- Project portfolio

- Contact section

## Project IT Support yang Ditampilkan

### 1. Helpdesk Ticketing System

Dokumentasi alur kerja helpdesk IT, mulai dari pencatatan tiket, kategori masalah, prioritas, SLA, status tiket, hingga laporan penyelesaian.

### 2. IT Asset Inventory

Template pencatatan aset IT seperti laptop, PC, printer, monitor, router, access point, dan perangkat lainnya.

### 3. Network Troubleshooting

Panduan troubleshooting jaringan LAN/WiFi untuk masalah internet, DHCP, DNS, IP conflict, WiFi lambat, dan akses printer/server.

### 4. Windows PC Deployment

Checklist standar instalasi komputer baru, mulai dari OS, driver, aplikasi kerja, antivirus, jaringan, printer, dan serah terima user.

### 5. Backup Automation

Dokumentasi konsep backup file/database, jadwal backup, struktur folder, retensi data, dan testing restore.

### 6. CCTV Network Setup

Dokumentasi konfigurasi CCTV berbasis jaringan, DVR/NVR, IP camera, storage rekaman, remote access, dan mapping titik kamera.
